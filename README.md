# 🧠 ENIGMA CWAS for resting-state fMRI

## Description
This repository provides code to perform a Connectome-Wide Association Study (CWAS) on connectivity matrices generated using HALFpipe. 

With just a few steps, this script automates CWAS across your pipelines, making it fast and reproducible.

## Requirements
Before you begin, make sure you have:

- Connectomes generated by HALFpipe (.csv files)
- A phenotype file (CSV, TSV or xlsx) containing subject IDs that match those in your connectome filenames

## 🚀 How to Run
1. Launch a Jupyter Notebook session.
2. Edit the params.py file:
- Set the path to your working directory and phenotype file
- Set the path to acces the altas description file (CSV file)
- Indicate where you want to save the results
3. In the notebook, specify:
- The column(s) in the phenotype file you want to analyze
- Any covariates or variables of interest

Then run the notebook cells to verify your working directory and start the CWAS.

## 📌 Notes
- This tool assumes the connectomes are subject-specific and have consistent naming.
- Ensure your phenotype file is cleaned and all required fields are filled.
- Results will typically include beta maps, pvalue maps and summaries.
